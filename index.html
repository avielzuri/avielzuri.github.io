Orbital Dynamics Simulator

Overview

A comprehensive Python-based satellite orbit simulation tool that provides 3D visualization of orbital mechanics. Users can customize orbital parameters to analyze how eccentricity, inclination, and altitude affect satellite motion.

Key Features

  • Interactive 3D orbit visualization with animated satellite tracking
  • Customizable orbital parameters (eccentricity, inclination, altitude)
  • Real-time physics simulation using differential equation solvers
  • Dynamic scaling for various orbital sizes
  • Educational tool for understanding orbital mechanics

Technical Implementation

  • Utilized SciPy's LSODA solver for accurate orbital calculations
  • Implemented Kepler's laws and Newtonian gravitation
  • Created smooth 3D animations using Matplotlib
  • Optimized performance for real-time visualization

EEG Motion Classification

Overview

NASA-funded research project developing brain-computer interfaces for controlled navigation tasks. The system processes EEG signals to classify different types of motion, enabling direct neural control of navigation systems.

Key Features

  • Real-time EEG signal processing and classification
  • Advanced signal processing pipeline using Python
  • Time-frequency analysis using Short-Time Fourier Transform (STFT)
  • Power Spectral Density estimation using Welch's method

Research Impact

  • Contributed to NASA-funded flight simulation research
  • Developed novel approaches to EEG-based navigation
  • Published findings to validate system performance
  • Advanced understanding of brain-computer interfaces
